# Local Archive

This folder holds files that are not used by the production website. Its contents are ignored by Git, except for this index.

- `source-media/` - original photos, videos, and duplicate source assets.
- `reference/` - downloaded sites and standalone component examples.
- `generated/` - presentations and inspection output created from the project.
- `legacy-homepage/` - components and media from the previous homepage implementation.

Active application files remain in `src/` and `public/`. Move a file out of this archive before importing it into the website.
