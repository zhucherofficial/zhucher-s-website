# Source Media

This folder keeps original media files that are not imported directly by the app.

- `project-originals/` - source versions of project cover images.
- `profile-originals/` - source profile photography.
- `hero-originals/` - source hero video or motion media.
- `wechat-photos/` - imported WeChat photos reserved for future use.

Active website assets live in `src/assets/` and `public/hero-media/`.
